# Projeto de busca de filmes
## Pré-requisitos
- Java 11+
- Maven ou Gradle
- JUnit para testes unitários

## Instruções

### Executar Testes Unitários
```mvn test```

### Compilar o projeto
```mvn clean install```

### Executar o JAR
#### Busca Filme
```java -jar search.jar <parâmetro>```
#### Criar Cache
```java -jar cache.jar```

### Gerar Javadoc
```mvn javadoc:javadoc```
